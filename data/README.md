# Data Acquisition Guide

Large raster files are not committed to this repository due to GitHub size limits.
Follow the steps below to reproduce the full dataset.

---

## 1. VIIRS Nighttime Lights (via Google Earth Engine)

1. Open [https://code.earthengine.google.com](https://code.earthengine.google.com)
2. Paste the contents of `../scripts/gee_export.js` into the Code Editor
3. Click **Run** — this will queue export tasks to your Google Drive
4. Wait for all tasks to complete (check the **Tasks** tab)
5. Download the exported files from `Google Drive / viirs_electrification/` to `data/raw/`

**Expected files per country per year:**
- `{Country}_NTL_{year}.tif` — VIIRS yearly median composite (1 km, float32)
- `{Country}_tiles_{year}.csv` — Tile-level feature table (avg_rad, population, built_surface, slope_deg)

---

## 2. Administrative Boundaries (GADM)

Download from [https://gadm.org/download_country.html](https://gadm.org/download_country.html)

| Country | File | Notes |
|---------|------|-------|
| Morocco | `gadm41_MAR_0.gpkg` | Includes Western Sahara in national boundary |
| Brazil  | `gadm41_BRA_0.gpkg` | National boundary |
| China   | `gadm41_CHN_0.gpkg` | National boundary |

Save to `data/boundaries/`.

---

## 3. WorldPop Population (optional — available via GEE in the export script)

If you prefer direct download:
- [https://www.worldpop.org/geodata/listing?id=69](https://www.worldpop.org/geodata/listing?id=69)
- Select "Unconstrained individual countries 2000-2020" → Morocco / Brazil / China
